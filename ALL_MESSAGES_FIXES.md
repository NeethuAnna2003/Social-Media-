## 🚀 HOW TO USE

### Sending a Text Message
1. Type in the input field
2. Press Enter or click Send button
3. Message appears instantly

### Sending a Voice Message
1. Click the **microphone icon** (inside input field, right side)
2. Speak your message
3. Click **Send button** (purple circle) to send
4. Or click **X** to cancel

### Sending Media
1. Click the **paperclip icon** (left of input)
2. Select image, video, or audio file
3. Optionally add text
4. Click Send

### Viewing Online Status
- **Green dot** = User is online
- **No dot** = User is offline
- Status updates in real-time

   ```bash
   venv\Scripts\activate
   python manage.py runserver
   ```

2. **Start the frontend:**
   ```bash

   npm run dev
   ```
