Welcome to Connectify API

A social networking platform with AI-powered features

API Endpoints

GET /api/health/
Check API health status

Authentication

POST /api/token/
Obtain JWT token (login)
POST /api/token/refresh/
Refresh JWT token
POST /api/token/verify/
Verify JWT token

User Accounts

POST /api/accounts/register/
Register a new user
GET PUT /api/accounts/profile/
Get or update user profile (authenticated)
GET /api/accounts/profile/<username>/
Get public profile
POST /api/accounts/follow/<username>/
Follow a user
POST /api/accounts/unfollow/<username>/
Unfollow a user
