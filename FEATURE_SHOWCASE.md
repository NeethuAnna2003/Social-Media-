## 🎭 3. Categorization Logic

### Flow Diagram
```
Article Received
      │
      ▼
┌─────────────────┐
│ Check Tech      │
│ Exclusions      │
│ (CPU, Intel,    │
│  Laptop, etc.)  │
└────┬────────────┘
     │
     ├─ YES → Keep as Technology
     │
     ▼ NO
┌─────────────────┐
│ Check Entertain-│
│ ment Keywords   │
│ (Movie, Netflix,│
│  Gaming, etc.)  │
└────┬────────────┘
     │
     ├─ YES → Force Entertainment
     │
     ▼ NO
┌─────────────────┐
│ Use NewsAPI     │
│ Category        │
│ (business, tech,│
│  sports, etc.)  │
└────┬────────────┘
     │
     ▼
┌─────────────────┐
│ Final Category  │
│ Assigned        │
└─────────────────┘
```

### Examples
```
✅ "Intel Announces New CPU"
   Keywords: intel, cpu
   → TECHNOLOGY (excluded from entertainment)

✅ "Netflix Releases Marvel Trailer"
   Keywords: netflix, marvel, trailer
   → ENTERTAINMENT (forced)

✅ "Apple Reports Q4 Earnings"
   NewsAPI: business
   → BUSINESS (no overrides)

✅ "Lakers Win Championship"
   NewsAPI: sports
   → SPORTS (no overrides)
```

## 💬 6. Discussion Moderation

### Moderation Flow
```
User Types Comment
      │
      ▼
┌─────────────────┐
│ Check Spam      │
│ Patterns        │
│ (URLs, "click   │
│  here", etc.)   │
└────┬────────────┘
     │
     ├─ SPAM → ❌ Reject
     │
     ▼ CLEAN
┌─────────────────┐
│ Check Toxicity  │
│ (hate, stupid,  │
│  offensive)     │
└────┬────────────┘
     │
     ├─ TOXIC → ❌ Reject
     │
     ▼ CLEAN
┌─────────────────┐
│ Check Duplicate │
│ (exact match)   │
└────┬────────────┘
     │
     ├─ DUPLICATE → ❌ Reject
     │
     ▼ UNIQUE
┌─────────────────┐
│ ✅ Post Comment │
└─────────────────┘
```

### Examples
```
❌ "Click here to buy cheap products!"
   → REJECTED: Spam pattern detected

❌ "This article is stupid and the author is an idiot"
   → REJECTED: Toxic language detected

❌ "Great article!" (posted twice)
   → REJECTED: Duplicate comment

✅ "Great article! Very informative."
   → ACCEPTED: Clean, unique comment
```

     │
     ├─ FOUND & FRESH → Display Cached
     │

     │
     ▼

     │
     ▼
